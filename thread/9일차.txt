 # 쓰레드 동기화.

동기화의 필요성.
여러 쓰레드에서 공유하는 자원을 동시에 사용하면 카운트가 되지 않는 문제가 발생한다.

동기화.
동시에 어떤 공유하는 자원을 사용할 때 누군가 사용하고 있으면 사용하지 못하도록 막는 것.



 # monitor lock을 이용한 동기화.

Critical Section 임계영역.
공유하는 자원 중에 동시에 사용할 수 없는 곳을 의미.

intrinsic lock or monitor lock.
임계영역을 들어갈 때 막을 수 있는 모든 객체가 가지고 있는 잠금 장치.

잠금방법.
synchronized ( Object ) { 
	동기화 시킬 코드 ;
}

주의사항.
Object는 기본형 타입을 제외한 모든 객체가 가능하다.
