 # 값의 진법 변환.

이진수 -> 십진수로 바꾸기.
0001 -> 1
0010 -> 2
0100 -> 4
1000 -> 8

십진수 -> 이진수로 바꾸기.
2로 계속 나누어본다.
6 -> 4 + 2 -> 0110
11 -> 8 + 2 + 1 -> 1011

4진수.
2진수를 2개씩 묶어서 표현할 때 사용. ( 0 ~ 3 )

8진수.
2진수를 3개씩 묶어서 표현할 때 사용. ( 0 ~ 7 )

16진수.
2진수를 4개씩 묶어서 표현할 때 사용. ( 0 ~ F로 표현 )



 # 정수 값의 형식명칭과 형식변환.

진수 표현방법.
2진수 : 0b 
10진수 : 안붙임
16진수 : 0x ( 0x1A )

정수형식.
byte : 8비트
short : 16비트
int : 32비트 
long : 64비트

명시적 형변환 - 강제형변환 cast.
byte x = 1010 ; ( 컴파일 오류 )
byte x = (byte)1010 ;
