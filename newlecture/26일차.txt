 # 비트연산자.

비트 연산자를 사용하는 경우.
메모리 공간이 넉넉하지 않아서 비트 하나하나 아껴서 사용해야 될 경우 사용한다.

비트 논리 연산자.
& : 비트 교집합 연산자.
| : 비트 합집합 연산자.
^ : 비트 차집합 연산자.
~ : 비트 부정 연산자.

비트 논리 연산자 예시.
3 & 7 => 0011 & 0111 => 0011
3 & 7 => 0011 & 0111 => 0111

쉬프트 연산자.
<< : 왼쪽 쉬프트 연산자.
>> : 오른쪽 쉬프트 연산자.

쉬프트 연산자 예시.
3 << 2 => 0011 << 2 => 1100 => 12
3 >> 2 => 0011 >> 2 => 0000 => 0

쉬프트 연산자 주의사항.
오른쪽 쉬프트 연산자는 부호비트가 0일 경우 0으로 채우고, 부호비트가 1일 경우 1로 채워진다.
