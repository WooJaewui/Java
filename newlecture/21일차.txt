 # printf 함수의 형식 지정자.

형식.
%d : 정수 10진수.
%x : 정수 16진수.
%o : 정수 8진수
%f : 실수
%e : 지수형 실수.
%c : 문자.
%s : 문자열.

포맷 문자열과 형식 지정자.
%d, %s 같은 형식에 매칭된 값이 형식으로 형변환이 안되면 Exception 발생.



 # 형식 지정자 옵션.

변환 코드.
%10d : width를 10으로 지정.
%-10d : width 10에서 flag 위치를 정한다.
%5.2f : width 5, 실수 부분 2자리. ( 62.32이 출력 )
%2$d : 뒤에 두번째 , 에 값과 매칭한다. ( index )
