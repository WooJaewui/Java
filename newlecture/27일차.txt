 # 복합 대입 연산자 / 삼항 연산자 / 연산자 우선 순위.

복합 대입 연산자.
+= : 덧셈 후에 대입.
-= : 뺄셈 후에 대입.
*= : 곱셈 후의 대입.
/= : 나눗셈 후에 대입.
%= : 나머지 연산 후에 대입.
&= : 비트 AND 후에 대입.
|= : 비트 OR 후에 대입.
>>= : 오른쪽 쉬프트 후에 대입.
<<= : 왼쪽 쉬프트 후에 대입.

삼항 연산자.
( 조건 ) ? value1 : value2 
조건이 true이면 value1을 , 조건이 false이면 value2를 리턴한다.

객체 형식 비교 연산자.
obj1 instanceof obj2 : obj1이 obj2의 자식클래스인지 확인한다.

연산자 우선순위.
단항연산자 -> 이항연산자.
외우지 말고 소괄호를 통해 우선순위를 직접 정해주자.
