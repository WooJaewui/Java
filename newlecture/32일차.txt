 # 컴퓨터 프로그램의 기본흐름과 그것을 제어하는 명령어의 종류.

조건문.
if : 조건이 참이면 실행하고 , 거짓이면 실행하지 않는다.
else if : if 조건이 거짓이고, else if 조건이 참이면 실행.
else : if와 else if 들이 모두 거짓이면 실행.
switch : case a , case b , ... 등의 조건에 따라 실행하는 코드가 달라짐.

반복문.
while : 조건이 false일 때까지 안에 코드를 반복한다.
for : 조건이 false가 될때까지 코드를 반복한다.

반복의 흐름을 조절하는 예약어.
continue : 반복문의 조건을 다시 확인하고 true이면 코드를 다시 시작한다.
break : 반복문을 종료한다.
